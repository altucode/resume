FallRecipes

An experiment in creating a simplified, functional and fall-themed copy of the recipe-sharing website AllRecipes. Created using server-side Ruby/Rails and client-side Javascript (Backbone and jQuery with EJS templates). Image hosting via Amazon's S3 service and the Paperclip gem for Ruby. Hosted on Heroku. Uses the FatSecret nutritional API. Features a custom search implementation for querying the Postgres database against multiple required/excluded terms in join tables (ie, for food allergies).
